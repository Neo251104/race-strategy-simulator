# Race Strategy Simulator

## Overview

This project contains race strategy implementations developed across four levels. Each level builds progressively, improving control, structure, and overall performance in the simulation.

---

## Development Environment

The project was developed using:

* PyCharm
* Visual Studio Code

These tools were used for coding, testing, and refining the race strategy.

---

## Level 1: Basic Structure

### Description

Introduces the fundamental race format.

### Features

* Valid JSON structure
* Lap and segment definitions
* Basic corner and straight setup

### Goal

Ensure the simulation runs without errors.

---

## Level 2: Speed Control

### Description

Adds speed management on straight segments.

### Features

* Target speeds (84–88 m/s)
* More controlled acceleration

### Goal

Improve consistency and reduce instability.

---

## Level 3: Braking Strategy

### Description

Introduces braking before corners for better handling.

### Features

* Defined braking distances
* Improved corner entry control

### Goal

Balance speed and safety.

---

## Level 4: Full Strategy (Consistent Execution)

### Description

Applies a complete, consistent strategy across all laps.

### Features

* Repeated lap configuration
* Stable performance across the race
* No pit stops

### Goal

Maximize reliability and predictability.

---

## Limitations

* No tyre degradation handling
* No fuel-based adjustments
* No pit stop strategy

---

## Future Improvements

* Adaptive braking and speed
* Tyre management
* Pit stop optimization
* Dynamic race conditions handling

---

## Conclusion

The four levels demonstrate a structured progression from a basic setup to a complete race strategy. The final approach prioritizes consistency and reliability, providing a strong foundation for further optimization.
